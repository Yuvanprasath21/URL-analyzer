Analyzing URL

This is a model webpage for a website which can able to give information about a URL by analyzing it on various functions and terms. The URL analyzer is capable of generating the trustability score of a URL, this value is generated with the conditions specified for evaluating a URL. This analyzer does not give the values by considering the reviews of the user or any other external party; it gives the value by parsing the given URL. The value generated by this analyzer is more accurate because it analyzes the website itself. The conditions by which a URL is analyzed are listed below:

  • Having IP address
  • Double slash redirecting
  • SFH (Suspicious File Handling)
  • Age of Domain
  • Iframes
  • DNS record
  • Google Index
  • Links pointing to a URL
  • Abnormal URL
